# Agent Recipes

These recipes are transcription rails: intent to API to proof surface. The
machine-readable source of truth is `traceability/agent_surface.yaml`; each
recipe below should stay backed by a compiling example, template, or test.

- `build-with-batpak.md` - shortest end-to-end path.
- `typed-event-store.md` - typed event append.
- `region-read.md` - bounded reads.
- `read-walk-evidence.md` - read evidence reports.
- `projection.md` - projection and projection-run evidence.
- `cursor-replay.md` - at-least-once cursor replay.
- `lossy-subscription.md` - lossy live push.
- `artifact-envelope.md` - canonical artifact envelopes.
- `attested-registry.md` - generic registry rows.
- `backup-envelope.md` - backup manifest evidence.
- `state-transition.md` - generic state transition evidence.
- `reservation-ledger.md` - abstract reservation ledgers.
- `platform-evidence.md` - platform profile probes.
- `anti-patterns.md` - wrong moves and replacements.
