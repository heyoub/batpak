# @batpak/generated

Auto-generated event types, Effect 4 schemas, and golden hex vectors for
[batpak](https://github.com/heyoub/batpak). Generated from the
substrate-owned manifest by `@batpak/codegen`; the source tree is fully
overwritten on every regeneration and CI verifies the output is
byte-identical. Do not edit by hand.

Most applications should install [`@batpak/sdk`](https://www.npmjs.com/package/@batpak/sdk),
which re-exports this package.
